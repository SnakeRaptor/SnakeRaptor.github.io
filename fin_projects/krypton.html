Projects - Krypton

This page is about Krypton: a library I created which provides basic file encoding and decoding abilities.

About Krypton

When I started working on a number of projects, I realised quickly that I was using the same methods for basic file handling between them. So, instead of rewriting the same code over and over again, I created a library. That library is called Krypton. You can use Krypton too, if you want. It's available on GitHub using the link below.

Krypton's Future

Ultimately, Krypton was thrown together pretty hastily and is a bit of a mess. So, I've decided instead of adding any new features to it, I'm going to start a new project to replace it. This project will be called Submerge, and you can read more about it here.
