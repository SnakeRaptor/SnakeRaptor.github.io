Projects - Great Generic Focus Rework

This page is about another of my current modding projects for Hearts of Iron IV

About GGFW

The Great Generic Focus Rework, or GGFW, is a mod I've created for Hearts of Iron IV. It aims to overhaul the game's generic focus tree and add new politcal paths, as well as make minor nations actually have a chance of becoming strong.

Progress Report

The Great Generic Focus Rework is technically complete. The game is fully playable as any nation which uses the tree. However, I'm currently working on testing the mod for bugs. I also need to make some tweaks to the balance of the mod.

Another change I'm considering still making is adding a non-aligned branch to the political tree. However, I doubt this will be in the initial release of the mod.

Availability

The mod isn't yet available to download, as it is still in the testing phase. However, I will add a link to download it from the Steam Workshop when it becomes available.
