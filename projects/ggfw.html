Projects - Great Generic Focus Rework

This page is about another one of my modding projects for Hearts of Iron IV.

What is "Great Generic Focus Rework"?

GGFW is a mod for Hearts of Iron IV which completely rebuilds the generic focus tree in-game. The base-game one hasn't been updated for quite some time, so I decided I'd do it so Paradox didn't need to! The mod isn't yet available, but when it is I'll add a link here.
