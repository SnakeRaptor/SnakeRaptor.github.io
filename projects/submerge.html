Projects - Submerge

This page is about Submerge, the successor to Krypton.

About Submerge

Submerge is the successor to Krypton, my original library for file handling and encoding. Instead of utilising just encoding, however, it also supports encryption.

Data Drowning

One of Submerge's features, besides what has already been mentioned, is data drowning. Effectively, data is drowned within a collection of false data, and can only be accessed if you have the Drown Key through a process called "Rescuing".
