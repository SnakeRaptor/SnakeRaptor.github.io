Basic Information

What is Krypton?

Krypton is a library of methods that can be used for basic file handling, as well as for encrypting and decrypting strings.


What encryption does Krypton offer?

Krypton offers basic encryption and decryption using the following formats available through C#'s Encoding class:

  • ASCII
  • Latin1
  • Unicode
  • UTF-8
  • UTF-32
Please note that these forms of encryption are not extremely secure and should only be used to stop a person from directly accessing a file's data.

Setup

Basic Setup

To use Krypton, you must first add the DLL file as a reference. If you are not using Visual Studio, check your IDE's documentation to see how to add project references. In all IDEs, the final step will be the same. If you are using Visual Studio, you can add a reference by following these steps:

  1. Go to Project > Add Project Reference > Browse then locate and select the Vega.Krypton.dll file on your computer.
  2. Click OK and wait for Visual Studio to add the reference.
  3. Finally, import the library to a file. If you are using C#, you can do this with the following code: using Vega.Krypton;

The KryptonType Enum

Before you can Krypton's encryption features, you will need to create an instance of the KryptonType enum. You will pass this in whenever you are creating an instance of the Krypton class which is covered later. To create, and configure, the KryptonType enum, use the following steps:

  1. Complete the setup process outline above. Otherwise you won't have access to the KryptonType enum!
  2. Create a new instance of the KryptonType enum in the same way you would with any other. In C#, this can be done with the following line of code: KryptonType kt;
  3. Choose the type of encryption you wish to use. This can be done by assigning a value to the newly created enum instance. The values that can be assigned are: A, L, U, U8 and U32. Each corresponds to an encryption type: ASCII, Latin1, Unicode, UTF-8 and UTF-32 respectively. In C#, the value can be assigned with the following code: kt = KryptonType.A;

The Krypton Class

All of Krypton's encryption functionality is held within its namesake class: Krypton. You can create an instance of the Krypton class in the same way you would any other class. However, a KryptonType enum must be passed into the object when it is created. In C#, this can be done using the following code: Krypton krypt = new Krypton(kt);

Standard File Handling

Whilst the primary function of Krypton is its encryption services, the library also contains operations for basic reading and writing to files.


The UFile Class

Standard file handling functions can be found inside the UFile class. You can access these by creating an instance of the class. In C#, this is done with the following code: UFile uf = new UFile()


Writing to Files

The UFile class comes with two methods for writing to files: AppendToFile and OverwriteToFile.
The AppendToFile method will add text onto the end of an existing file, leaving the rest of the file intact.
The OverwriteToFile method will erase all text in a file and replace it with with the text passed into it.
Both methods require two arguments in the following order:

  1. The directory of the file you wish to write to.
  2. The text you wish to write to the file.


Reading from Files

In addition to writing to files, the UFile class comes with two methods for reading from files: ReadFromFile and ListFromFile.
The ReadFromFile method will return a string containing all of the text in the file within the given directory.
The ListFromFile method will return a List of string, where each item is a new line. This is especially useful when reading from a CSV, or other basic data storage file.
Both methods only reuire one argument: the directory of the file to be read from. Both methods will return null if an error occurs, so it is good practice to check the output from the method is not null.

Encryption and Decryption

The primary function of Krypton is to provide basic encryption to strings. Please note that these methods of encryption are not completely secure, and should only be used to make text not directly readable.


The Krypton Class

As mentioned before, all of Krypton's encryption and decryption functions are located in the Krypton class. Ensure you have set up an instance of the class before continuing.

The Encode Method

This method is used to encrypt a string which is passed to it. To use it in C#, you can use the code: krypt.Encode(myString);
From here, the string is taken and converted into the format you chose when creating the Krypton object. The method then returns the encrypted string.

The Decode Method

This method is used to decrypt a string which is passed to it. To use it in C#, you can use the code: krypt.Decode(myEncryptedString);
From here, the string is taken and converted from the format specified when creating the Krpyton object, into a string. This string is then returned.
Note that you need to know the type of encryption used for this method to work correctly.

The SwitchType Method

This method is used if you wish to change the type of encryption being used by the Krypton object. To use it, create a new KryptonType enum and call the method, passing the enum as an argument.
In C#, this can be done like this: krypt.SwitchType(kt);

File Handling with Encryption

File handling with encryption works similarly to standard file handling and will likely be the most useful feature.


The KFile Class

File handling functions with encryption can be found inside the KFile class. You can access these by creating an instance of the class. In C#, this is done with the following code: KFile kf = new KFile()


Writing to Files

Similarly to the UFile class, the KFile class contains two methods for writing to files:
The AppendToFile method will encrypyt text, using the chosen encoder, then add it to the end of a file leaving the previous contents intact.
The OverwriteToFile method will erase all text in a file and replace it with the text passed to the method, encrypted using the chosen encoder.
Both methods require three arguments in the following order:

  • The directory of the file you wish to write to.
  • The text you wish to write to the file (not encrypted).
  • A KryptonType enum configured to the desired encryption type.


Reading from Files

For reading from an encrypted file to work, you must know what type of encryption was used originally. The KFile class contains two methods for reading encrypted files.
The ReadFromFile method will read in text from the specified file, decrypt it using the chosen encryption method, then return a string of the decrypted text.
The ListFromFile method will read in text one line at a time into a List of strings, decrypting each line, then return the List of lines.
Both methods require two arguments in the following order:

  • The directory of the file to be read and decrypted.
  • A KryptonType enum confgirued to the desired encryption method.
